1- Installez le node fournis
2- Ouvrez un invit de commande "cmd.exe" et positionnez vous dans les documents de la session en cours

Puis tapez les commandes suivantes

$ mkdir json-server-lib
$ cd json-server-lib
$ npm init -y
$ npm i -g json-server

3- Créez un fichier db.json dans le dossier json-server-lib
4- Ajoutez le json du character à l'intérieur
5- Tapez la commande "json-server --watch db.json" dans votre invit de commande pour lancer le serveur

6- Si le serveur démarre il vous donne l'url, ouvez un navigateur et vérifiez que vos données soient bien présentes
7- Accédez à ces données via le projet à compléter pour qu'il fonctionne correctement
